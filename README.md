# today in space.
 
a website that fetches nasa's astronomy picture of the day! it's powered by nasa's API.
---
 
## what it does
 
- displays the title, explanation, and a picture of nasa's astronomy picture of the day.
- date picker to browse the full archive back to 1995
- random day button where it holds upto 30 years of images, you can click a button it picks one at random
